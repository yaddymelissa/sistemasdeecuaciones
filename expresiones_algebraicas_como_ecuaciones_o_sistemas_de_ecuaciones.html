Expresiones algebraicas como ecuaciones o sistemas de ecuaciones

Las  expresiones  algebraicas  se  usan  en   la   resolución   de   problemas planteando  los  postulados  numéricos  como  ecuaciones  o  sistemas  de ecuaciones, organizando los datos obtenidos y asignando letras a cada uno de ellos. Veamos:

  • La suma de dos números es 94. Si el mayor excede al menor en 14 unidades hallar los números.

m = número mayor                n = número menor

 Las ecuaciones serían

m + n = 94

m - 14 = n

  • Raúl tiene 12 años menos que León y la suma de sus edades es 30 años. ¿Cuál es la edad de cada uno?

Edad de León = O              Edad de Raúl = P

Ecuaciones:

P = O - 12

O + P = 30

  • Hallar 3 números enteros consecutivos cuya suma sea 108.

Para   muchos   ejercicios  de  este  tipo   se   pueden   usar  expresiones algebraicas en términos de una sola incógnita.

Como vimos anteriormente sería:

X = primer número.

 X + 1 = segundo número

X + 2 = tercer número

Ecuación: X + (X + 1) + (x + 2) = 108
