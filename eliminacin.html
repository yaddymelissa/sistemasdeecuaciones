Eliminación

Este método consiste en eliminar una incógnita de la siguiente manera

Paso 1. Multiplicar los dos miembros de una ecuación por un número conveniente, de modo que una de las incógnitas aparezca con coeficientes opuestos en las dos ecuaciones.

            Ejemplo:

                      

           Multiplicamos la ecuación  (1) por 2 y el sistema queda:

                    

Paso 2. Sumar miembro a miembro las dos ecuaciones para obtener otra con una sola incógnita.

 8X - 2Y = 28
 5X + 2Y = 11
13X + 0  = 39
           X = 39/13

           X = 3

Paso 3.  Reemplazamos el valor de X en la ecuación (2):

  5X + 2Y = 11

5(3) + 2Y = 11

  15 + 2Y = 11

          2Y = 11 - 15

           Y  = - 4/2

           Y  = - 2

EVALÚO MIS AVANCES

Practica este método con el sistema de ecuaciones (Completa el procedimiento):

                          5x + 2y = 23     (1)

                          7x - 3y = -20    (2)

Vamos a eliminar la variable Y, escribe cuáles son los números convenientes para multiplicar cada una: 

                          5x + 2y = 23     (1) multiplicamos por JXUwMDZi

                          7x - 3y = -20    (2) multiplicamos por JXUwMDZh

Multiplica las dos ecuaciones, por dichos números y realiza la suma miembro a miembro:

                              JXUwMDY5JXUwMDA0JXUwMDRkJXUwMDUzJXUwMDFkJXUwMDRm = JXUwMDZlJXUwMDBm

                             JXUwMDY5JXUwMDA1JXUwMDRjJXUwMDU1JXUwMDFiJXUwMDRm = JXUwMDc1JXUwMDE5JXUwMDA0

                               JXUwMDZhJXUwMDBiJXUwMDQx  + JXUwMDY4 = JXUwMDZhJXUwMDBi

                                                   x = JXUwMDZhJXUwMDBiJXUwMDE2JXUwMDFkJXUwMDBi

                                                   x = JXUwMDY5

Reemplaza el valor de X en la ecuación (1):

                      5x +  2y = 23

                5( JXUwMDY5 ) + 2y = 23

                   JXUwMDZk + JXUwMDZhJXUwMDRi = 23

                                2y = JXUwMDZhJXUwMDAxJXUwMDFlJXUwMDE4

                             JXUwMDZhJXUwMDRi = JXUwMDY5JXUwMDA5

                                 JXUwMDIx = JXUwMDY5JXUwMDA5JXUwMDE3JXUwMDFk

                                  y = JXUwMDYx

Te queda como ejercicio verificar que los valores encontrados cumplen con las igualdades del sistema.
