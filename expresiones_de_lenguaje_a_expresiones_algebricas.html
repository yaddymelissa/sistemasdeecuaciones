Expresiones de lenguaje a expresiones algebráicas

 

  • Tomemos n como cualquier número entero y simbolicemos algebráicamente:
    • El número anterior a n sería: n - 1
    • El número siguiente de n sería: n + 1
    • El triple de n se representa con: 3n
    • Dos números consecutivos siguientes a n: en este caso se trata de representar con n el primer número y los siguientes sumando luego 1, 2, 3... respectivamente: n, n + 1, n + 2, n+ 3....
  • Las expresiones algebraicas también se usan para representar relaciones. Tomemos   a   X   y   Y   como   números   reales   y   X   >   Y. Ahora representemos algebráicamente la relación que se establece en cada caso.
    • El doble del mayor es igual al triple del menor: 2X = 3Y
    • El mayor excede al menor en 5: X = Y + 5
    • El menor es igual al mayor disminuido en 12: Y = X-12
